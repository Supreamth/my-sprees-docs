1. ภาพรวมระบบ

ผู้ใช้เรียก CLI, parser รับ flags/values, normalize input, summarize, optional rounding แล้วส่งออกเป็น text / JSON / CSV

flowchart TD
    User[User / CLI caller] --> CLI[sandbox-math console script]
    CLI --> Parser[argparse parser]
    Parser --> Mode{Output mode?}
    Parser --> Values[values positional args]
    Parser --> Precision[--precision optional]

    Values --> Normalize[normalize_numbers]
    Normalize --> Summary[summarize]
    Summary --> Stats[Summary dictionary]
    Stats --> Round[round_summary if --precision]

    Round --> Mode
    Mode -->|default| Text[Python dict text output]
    Mode -->|--json| JSON[JSON output]
    Mode -->|--csv| CSV[2-line CSV output]

    Summary --> Fields[count / total / mean / minimum / maximum / median / range / unique / stddev]
Library + CLI

sandbox_math.py เป็นทั้ง module และ entry point

Data normalization

normalize_numbers() แปลง input เป็น float และ reject ค่าที่ไม่ใช่ตัวเลข

Output modes

รองรับ default text, --json, และ --csv

3. Data flow ของ CLI

ตัวอย่าง flow: sandbox-math --csv --precision 2 1 2 3.5

sequenceDiagram
    participant U as User
    participant C as sandbox-math CLI
    participant P as argparse
    participant N as normalize_numbers
    participant S as summarize
    participant R as round_summary
    participant O as Output formatter

    U->>C: sandbox-math --csv --precision 2 1 2 3.5
    C->>P: parse flags and values
    P-->>C: mode=csv, precision=2, values=[1,2,3.5]
    C->>N: normalize values
    N-->>C: [1.0, 2.0, 3.5]
    C->>S: summarize(numbers)
    S-->>C: summary dict with 9 fields
    C->>R: round_summary(summary, 2)
    R-->>C: rounded summary dict
    C->>O: format_csv(summary)
    O-->>U: CSV header + values

5. CI / verification flow

Workflow ที่ใช้คือ branch จาก main → TDD → local verify → push → PR → GitHub Actions → merge → cleanup

flowchart TD
    Dev[Developer / Agent] --> Branch[Create branch from main]
    Branch --> TDD[Write failing tests first]
    TDD --> Impl[Implement minimal code]
    Impl --> LocalTests[python3 -m unittest -v]
    LocalTests --> Smoke[Installed package smoke test]
    Smoke --> Push[Push branch to GitHub]
    Push --> PR[Open PR in GitHub Web]
    PR --> Actions[GitHub Actions: Python unittest + installed CLI smoke]
    Actions --> Merge[Merge PR into main]
    Merge --> Cleanup[Delete remote/local branch]
    Cleanup --> Final[Sync main + final verify]
CI checks

install package, unittest, help/version smoke, JSON/CSV/precision smoke

Local verification

python3 -m unittest -v และ installed CLI smoke test

Known limitation

เครื่อง local ไม่มี gh/API token จึงต้องเปิด/merge PR ผ่าน GitHub Web
